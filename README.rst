Requeriments
============

**Linux**

    * Python 2.7.x
    * Git

**OS-X**

    * Brew
    * Python 2.7.x
    * Git

**Microsoft Windows**

    * Python 2.7.x
    * Git

Installation
============

**Linux**

    **Debian**

        .. code-block:: bash

           $ sudo apt-get update
           $ sudo apt-get install python-dev
           $ sudo apt-get install git

    **Centos**

        .. code-block:: bash

           $ yum -y update
           $ yum groupinstall -y 'development tools'

**OS X**

    .. code-block:: bash

        $ brew install python
        $ brew install git

**Microsoft**

    .. code-block:: bash

        $ http://git-scm.com/downloads
        $ https://www.python.org
